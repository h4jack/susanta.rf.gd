Getting the Cursor Position
To get the cursor position inside a contenteditable div, you can use the Selection object in JavaScript:

function getCursorPosition() {
    let cursorPos = { start: 0, end: 0 };
    let selection = window.getSelection();
    if (selection.rangeCount > 0) {
        let range = selection.getRangeAt(0);
        cursorPos.start = range.startOffset;
        cursorPos.end = range.endOffset;
    }
    return cursorPos;
}
In this function:

window.getSelection() gets the current selection object.
selection.getRangeAt(0) retrieves the range of the current selection.
range.startOffset and range.endOffset give you the start and end positions of the selection within the contenteditable div.
Getting Selected Text
To get the selected text, you can modify the function to return the selected text instead of just the cursor position:

function getSelectedText() {
